RITM Ticket Formatter

Reads a ServiceNow RITM page and formats it as a polished Jira ticket. One click to copy rich-text content ready to paste into Jira.

ServiceNow Jira Live Events

How to install

1
Click the button above — a .zip file will download
2
Extract the ZIP, then open chrome://extensions
3
Enable Developer mode (top-right toggle)
4
Click Load unpacked and select the ritm-ext folder
5
Open any RITM in ServiceNow and click the 🎫 icon
📺

TXEdge VLC Launcher

Detects SRT output streams on any TXEdge page and launches them directly in VLC with a single click. Passphrase stored securely in Chrome.

TXEdge SRT VLC Streaming

How to install

1
Click the button above — a .zip file will download
2
Extract the ZIP, then open chrome://extensions
3
Enable Developer mode (top-right toggle)
4
Click Load unpacked and select the txedge-ext folder
5
On first use, enter your SRT passphrase — saved locally, never leaves Chrome
🔬

SO Video Analyser for Chrome

Detects SRT streams on any TXEdge/TXCore page and triggers a full video analysis via the SP SO Proxy — one click, result inline. Proxy URL and passphrase saved once, automatic every time.

TXEdge SRT GOP Analysis

How to install

1
Click the button above — a .zip file will download
2
Extract the ZIP, then open chrome://extensions
3
Enable Developer mode (top-right toggle)
4
Click Load unpacked and select the so-video-analyser folder
5
On first use, enter the Proxy URL and your SRT passphrase — saved locally, never leaves Chrome
