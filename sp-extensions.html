📖 How to install (3 steps, 1 minute)
1

Download & unzip the extension

Click Download & Install on the card above. A .zip file is saved to your Downloads folder. Right-click it and choose Extract All…

2

Open Chrome Extensions

In your browser, navigate to chrome://extensions and toggle Developer mode ON (switch in the top-right corner).

💡 Developer mode lets Chrome load extensions from your local folder instead of the Web Store.
3

Load the extension folder

Click Load unpacked (top-left button). Browse to the unzipped folder — the one that contains manifest.json — and click Select Folder.

✅ Done! The 🎫 icon will appear in your Chrome toolbar. Navigate to any ServiceNow RITM and click it.
🔄

Updating the extension

When a new version is released, download and replace the folder contents, then go to chrome://extensions and click the ↻ refresh icon on the extension card.
